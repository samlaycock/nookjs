This is Javascript library that implements a Javascript code interpreter.
This project uses Bun as the package manager.

For how to contribute to the codebase, see conventions/CONTRIBUTING.md
For how to go about fixing bugs, see conventions/BUG_FIXES.md
For code style conventions, see conventions/CODE_STYLE.md
For testing conventions, see conventions/TESTING.md
For TypeScript conventions, see conventions/TYPESCRIPT.md
For feature-specific behavior and interpreter gotchas, see the relevant file in docs/

If you need to keep track of anything with a file on disk, create a .txt file
and write your thoughts in markdown format.
